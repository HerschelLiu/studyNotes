## bash(命令行)

```bash
npm init // 初始化npm
// 安装新版本
npm install --save-dev webpack
// 安装特定版本
npm install --save-dev webpack@<version>
// 如果你使用 webpack 4+ 版本，你还需要安装 CLI。
npm install --save-dev webpack-cli
// 简写
npm install --save-dev webpack webpack-cli
```


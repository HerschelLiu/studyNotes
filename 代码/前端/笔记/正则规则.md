### 金额

```js
const reg = /^(0|[1-9]\d*)(\.\d{1,2})?$/

// 输入过滤
let value = input
    .replace(/[^\d.]/g, '') // 去除非数字和小数点[7](@ref)
    .replace(/(\..*)\./g, '$1') // 禁止多个小数点
    .replace(/^0+(\d)/, '$1') // 防止前导零（如 00123 → 123）
    .replace(/^\./, '') // 禁止小数点开头
    .replace(/(\.\d{2})\d+/, '$1'); // 限制两位小数
```

1. `0`：允许输入单个0；
2. `[1-9]\d*`：非零整数部分，不能以0开头；
3. `(\.\d{1,2})?`：可选的小数部分，最多两位；
4. 整体匹配金额如 `0`、`0.5`、`0.50`、`123`、`123.45` 等
